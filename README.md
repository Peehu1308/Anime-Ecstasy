// Anime Ecstasy – A React + Tailwind CSS SPA

📌 Summary
Anime Ecstasy is a blazing-fast, responsive anime browsing website built using React + Vite + Tailwind CSS.
Users can explore trending anime, view detailed info, and manage their personal watchlist.

🚀 Features
- 🎴 Trending Anime Gallery (via Jikan API)
- 📄 Detailed Anime View
- ➕ Watchlist saved in localStorage
- 🔄 Pagination support
- ⚡ Vite dev server + build
- 📱 Responsive Tailwind UI

🧱 Stack
React | Vite | Tailwind CSS | Axios | localStorage | Jikan API

📁 Folder Structure
anime-ecstasy/
├── public/
├── src/
│   ├── assets/
│   ├── components/
│   ├── App.jsx
│   └── main.jsx
├── tailwind.config.js
├── CONTRIBUTION.md
└── README.md

🧑‍💻 To Run
git clone https://github.com/yourusername/anime-ecstasy.git
cd anime-ecstasy
npm install
npm run dev

📦 Before Contributing
→ Read CONTRIBUTION.md before creating a PR

🎯 This project uses Tailwind CSS and is built as a single-page application (SPA) using React and Vite.
